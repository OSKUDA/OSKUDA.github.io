Describes the part of the user interface represented by this widget.

The framework calls this method when this widget is inserted into the tree in a given BuildContext and when the dependencies of this widget change (e.g., an InheritedWidget referenced by this widget changes). This method can potentially be called in every frame and should not have any side effects beyond building a widget.

The framework replaces the subtree below this widget with the widget returned by this method, either by updating the existing subtree or by removing the subtree and inflating a new subtree, depending on whether the widget returned by this method can update the root of the existing subtree, as determined by calling Widget.canUpdate.

Typically implementations return a newly created constellation of widgets that are configured with information from this widget's constructor and from the given BuildContext.

The given BuildContext contains information about the location in the tree at which this widget is being built. For example, the context provides the set of inherited widgets for this location in the tree. A given widget might be built with multiple different BuildContext arguments over time if the widget is moved around the tree or if the widget is inserted into the tree in multiple places at once.

The implementation of this method must only depend on:

  • the fields of the widget, which themselves must not change over time, and
  • any ambient state obtained from the context using BuildContext.dependOnInheritedWidgetOfExactType.

If a widget's build method is to depend on anything else, use a StatefulWidget instead.

See also:

  • StatelessWidget, which contains the discussion on performance considerations.

@override
Widget build(BuildContext context) {
  return Scaffold(
    body: BlocBuilder<JobScreenCubit, JobScreenState>(
      builder: (context, state) {
        if (state is JobScreenLoading) {
          return const Center(child: CircularProgressIndicator());
        }

        if (state is JobScreenLoaded || state is JobScreenEmpty) {
          var _jobs = context.watch<JobScreenCubit>().state.jobList;

          return SafeArea(
            child: SizedBox(
              child: ContainedTabBarView(
                tabBarProperties: TabBarProperties(
                  indicator: BoxDecoration(
                    borderRadius: const BorderRadius.only(
                      topLeft: Radius.circular(14),
                      topRight: Radius.circular(14),
                    ),
                    shape: BoxShape.rectangle,
                    color: Theme.of(context).primaryColor,
                  ),
                  indicatorPadding: EdgeInsets.only(top: 48.h),
                  height: 54.h,
                  indicatorSize: TabBarIndicatorSize.label,
                  labelColor: Theme.of(context).primaryColor,
                  unselectedLabelColor: Colors.black.withOpacity(0.65),
                ),
                tabs: [
                  Text(
                    'Open',
                    style: TextStyle(
                      fontWeight: FontWeight.bold,
                      fontSize: 15.sp,
                    ),
                  ),
                  Text(
                    '''Ongoing (${_jobs.where(
                          (element) =>
                              element.jobStatus?.status == 'Ongoing' ||
                              element.jobStatus?.status == 'Accepted' ||
                              element.jobStatus?.status == 'Resceduled',
                        ).length})''',
                    style: TextStyle(
                      fontWeight: FontWeight.bold,
                      fontSize: 15.sp,
                    ),
                  ),
                  Text(
                    'Past',
                    style: TextStyle(
                      fontWeight: FontWeight.bold,
                      fontSize: 15.sp,
                    ),
                  ),
                ],
                initialIndex: 0,
                views: [
                  // Open Tab
                  if (_jobs.isEmpty) const EmptyJobScreen(),
                  if (_jobs.isNotEmpty)
                    JobPostListWidget(
                      jobs: _jobs
                          .where((element) =>
                              element.jobStatus?.status == 'Open')
                          .toList(),
                      parentTab: JobScreenTabs.open,
                    ),
                  // Ongoing tab
                  if (_jobs
                      .where((element) =>
                          element.jobStatus?.status == 'Ongoing' ||
                          element.jobStatus?.status == 'Accepted' ||
                          element.jobStatus?.status == 'Resceduled')
                      .toList()
                      .isEmpty)
                    const EmptyJobScreen(
                      heading: 'No Job Started Yet',
                      para:
                          '''Hire a Pro to start a job OR Tap ‘Start Posting’ button below to post a new job and start getting offers from the people for work.''',
                    ),
                  if (_jobs
                      .where((element) =>
                          element.jobStatus?.status == 'Ongoing' ||
                          element.jobStatus?.status == 'Accepted' ||
                          element.jobStatus?.status == 'Resceduled')
                      .toList()
                      .isNotEmpty)
                    JobPostListWidget(
                      jobs: _jobs
                          .where((element) =>
                              element.jobStatus?.status == 'Ongoing' ||
                              element.jobStatus?.status == 'Accepted' ||
                              element.jobStatus?.status == 'Resceduled')
                          .toList(),
                      parentTab: JobScreenTabs.ongoing,
                    ),
                  // Past tab
                  if (_jobs
                      .where((element) =>
                          element.jobStatus?.status == 'Cancelled' ||
                          element.jobStatus?.status == 'Completed')
                      .toList()
                      .isEmpty)
                    const EmptyJobScreen(
                      heading: 'No Job Completed Yet',
                      para:
                          '''Comeback here when a job gets completed. OR Tap ‘Start Posting’ button below to post a new job and start getting offers from the people for work.''',
                    ),
                  if (_jobs
                      .where((element) =>
                          element.jobStatus?.status == 'Cancelled' ||
                          element.jobStatus?.status == 'Expired' ||
                          element.jobStatus?.status == 'Completed')
                      .toList()
                      .isNotEmpty)
                    JobPostListWidget(
                      jobs: _jobs
                          .where((element) =>
                              element.jobStatus?.status == 'Cancelled' ||
                              element.jobStatus?.status == 'Expired' ||
                              element.jobStatus?.status == 'Completed')
                          .toList(),
                      parentTab: JobScreenTabs.past,
                    ),
                ],
                onChange: print,
              ),
            ),
          );
        }
        return Container();
      },
    ),
  );
}
