JobDetails

JobDetails class Null safety

Model class to manage job details of a particular Job instance of the user

Holds multiple job details for each kind of service : Pet,Appliance, Cook, Gardening, VehicleWash, Tailor, Mason, Sanitation, Painter, Carpenter
